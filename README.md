# conditional_sentences
Данный код использовался в моей курсовой работе на тему "Функционально-семантическое поле условия в английском языке". Код позволяет извлечь условные предложения из корпусов Brown и Gutenberg, а также первой 1000 статей из Википедии. Далее он производит их подсчет в художественных и научных произведениях, создает сравнительную таблицу и диаграмму, а также дает примеры каждого типа условных предложений.
